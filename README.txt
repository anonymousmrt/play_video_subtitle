# MP4 Player with Subtitle

## Hướng dẫn sử dụng
1. Mở file `index.html` bằng trình duyệt.
2. Chọn file video MP4 và file phụ đề (.vtt, .srt, hoặc .crt).
3. Nếu phụ đề là .crt hoặc .srt, trang sẽ tự động chuyển sang định dạng .vtt để trình duyệt phát được.

**Lưu ý:**
- Nếu file .crt không phải định dạng SubRip (giống .srt), vui lòng gửi ví dụ nội dung để tôi hỗ trợ chuyển đổi.
- Trang này không upload dữ liệu ra ngoài, chỉ chạy trên máy bạn.
